# Mind Connection Reveal

A cinematic reveal app built with HTML, CSS, JS and a small Node backend.

## Setup

1. Place your director-ready blank-card video in `public/video.mp4`
   - The app overlays the chosen number onto the card in real time.
   - Use a 16:9 cinematic clip for best effect.

2. Install dependencies

```bash
npm install
```

3. Start the app

```bash
npm start
```

4. Open:

```
http://localhost:3000
```

## How it works

- The user enters a two-digit number.
- The browser sends the number to the backend.
- The page loads the cinematic video and overlays the chosen number on the card.
- The reveal text is updated to make it feel impossibly connected.
